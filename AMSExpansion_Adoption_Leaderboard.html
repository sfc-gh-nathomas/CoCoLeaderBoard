Data Definitions:
Accts Using = Number of accounts with product activity (CoCo or SI) in the pair's portfolio.
Total Accts = Total accounts assigned to the AE/SE or DM/SEM pair in AMSExpansion.
% CoCo / % SI = Adoption rate — Accts Using / Total Accts.
SI Users (90d) = Distinct users with Snowflake Intelligence activity in the last 90 days.
SI WAU = Weekly Active Users — distinct SI users in the last 7 days.
SI Credits = Token credits consumed by SI queries in the 90-day window.
SI Questions = Distinct requests (queries) made to Snowflake Intelligence.
CoCo Credits = Cortex Code credits consumed in the last 90 days. Includes CLI, Desktop, and UI usage.
Region / District / Patch = From SE Attributes hierarchy.
RVP = Regional Vice President (sales). SE Director = SE leadership counterpart.
Open Position Exclusion = Only accounts owned by active Account Executives are included.
